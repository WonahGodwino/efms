# Enterprise Financial and Logistics Management System

Enterprise Financial and Logistics Management System (EFMS) for multi-subsidiary operations, approvals, reporting, and analytics.

## Overview

This repository contains:
- Backend API and business logic in `backend/`
- Frontend web application in `frontend/`
- Infrastructure and support assets in `docker/`

## Quick Start

1. Install backend dependencies: `cd backend && npm install`
2. Install frontend dependencies: `cd frontend && npm install`
3. Configure environment variables for backend and frontend.
4. Run backend and frontend locally with npm scripts.

## Notes

- Supports multi-subsidiary workflows.
- Role-based access is implemented across backend and frontend.
- See open issues for architecture hardening tasks around company-level tenant scope.
